# 2023南京农业大学计算机视觉课程设计――手写汉字识别

## 使用说明：

1. 测试时，请修改`test.py`中的`test_dir`的值，改为test数据所在文件夹路径

2. 修改后直接运行`test.py`即可测试模型性能

3. 控制台会输出准确率与损失值

4. logs文件夹内记录了各种训练过程
   - `_pretrained`为使用了预训练模型
   - `_adam`为使用`adam`优化器，如果没有该后缀则为使用SGD
   - `csv`文件为训练的过程中准确率与loss的变化，还有训练时长
   
5. 由于模型参数文件过大，此处仅保留了最优的`resnext101_pretrained_40.pt`，在`model`文件夹内，其余训练结果截图在此

   ![model示例](.\model示例.png)

5. latex文件夹内为报告源码
